# 基本概念

## 什么是信息图

**信息图（Infographic）** 是一种将数据、信息和知识通过图形化方式进行可视化呈现的表达形式。它结合了图形设计、数据可视化和信息传达的优势，用直观、美观的视觉元素来简化复杂信息，帮助受众快速理解和记忆关键内容。

相比传统的文本说明或数据表格，信息图更强调视觉叙事能力，通过图形、图标、色彩、布局等设计元素，将抽象的数据转化为具象的视觉语言。

## 信息图的核心价值

### 提升信息传达效率

人类大脑处理视觉信息的速度是文字的 60,000 倍。信息图利用这一认知特点，通过图形化呈现，让受众能够在短时间内获取和理解大量信息。

### 增强记忆留存

视觉化的信息比纯文本更容易被记忆。研究表明，人们在 3 天后能记住 65% 的视觉信息，而对于纯文本信息的记忆率仅为 10%。

### 简化复杂概念

通过图形化的比喻和隐喻，信息图能将复杂的流程、关系或数据转化为易于理解的视觉模型，降低认知负担。

### 提升内容吸引力

精心设计的信息图具有较强的视觉吸引力，能够激发受众的阅读兴趣，提高内容的传播效果。

## 信息图与传统图表的区别

| 维度         | 传统图表         | 信息图                   |
| ------------ | ---------------- | ------------------------ |
| **主要目的** | 精确展示数据     | 叙述故事、传达洞察       |
| **设计重点** | 功能性、准确性   | 视觉美感、叙事性         |
| **表达方式** | 标准化图表类型   | 自由创意布局             |
| **信息密度** | 专注于特定数据集 | 整合多维度信息           |
| **受众定位** | 专业分析人员     | 大众传播                 |
| **视觉元素** | 以轴线、网格为主 | 图标、图形、装饰元素丰富 |

## 信息图的基本特征

### 视觉优先

信息图的核心是视觉传达。根据设计准则，**图形总占比应控制在 60% 左右**，文本占比约 40%，以充分发挥可视化的优势。在一些强调整体感的设计中，图形占比甚至可高达 80%。

### 结构化组织

一个完整的信息图通常包含以下基本元素：

- **标题（Title）**：概括主题，吸引注意
- **描述（Description）**：补充说明，提供上下文
- **数据项（Items）**：核心信息单元
- **图形元素（Shapes）**：装饰、强化视觉效果
- **图标（Icons）**：语义化的视觉符号
- **连接元素**：箭头、线条等，建立信息关系

### 语义清晰

信息图应遵循**中性化设计准则**，使用语义明确、国际通用的图形符号，避免情感化或文化特异性元素，确保信息传达的准确性和普适性。

### 层次分明

通过色彩深浅、图形大小、空间布局等手段建立清晰的视觉层次，引导受众按照设计者预期的顺序阅读和理解信息。

## 信息图的应用场景

### 数据报告

将枯燥的数据报告转化为易读的可视化内容，如年度总结、市场分析、用户调研报告等。

### 流程说明

展示业务流程、操作步骤、发展历程等时序性或逻辑性信息。

### 知识科普

将专业知识转化为通俗易懂的可视化内容，降低学习门槛。

### 对比分析

通过视觉化方式展现多个选项、方案或产品之间的差异与优劣。

### 关系展示

呈现组织架构、思维导图、因果关系等复杂的结构化信息。

### 时间轴

展示历史事件、项目进度、产品迭代等时间相关的信息。

## 设计哲学

@antv/infographic 致力于让数据叙事更简单、更优雅、更高效。我们认为：

- **简洁胜于复杂**：避免过度装饰，让信息本身成为主角
- **清晰胜于炫技**：追求信息传达的准确性，而非仅仅视觉效果
- **专业与美感并重**：在保证专业性的同时，提供令人愉悦的视觉体验
- **标准化与灵活性平衡**：提供开箱即用的模板，同时支持深度定制

通过理解这些基本概念，你将能更好地运用信息图进行数据叙事，让复杂信息变得简单易懂。
